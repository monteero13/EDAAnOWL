EDAAnOWL Ontology (Khaos UMA) — aligned with IDS Information Model: Description back to ToC

Extends IDS to describe Data Apps, Data Profiles, Metrics, Offers/Policies and their fit with BIGOWL (Algorithms, Problems, Workflows).
